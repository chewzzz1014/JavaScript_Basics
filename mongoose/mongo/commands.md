# Mongo Shell Commands

## Basic Navigation
   - show dbs
     - Show all database on the local mongo server
   - use <db name>
     - Switch to a database. Create new one if it doesn't exist
   -show collections
    - Show all collections in a database.
   - db
     - Show current database
   - cls
     - Clear screen
   - exit
     - Exit the shell
